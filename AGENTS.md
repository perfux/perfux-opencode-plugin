# Perfux Plugin

## Available Tools

### @research agent
Use for external research - docs, patterns, libraries. Has access to:
- Context7 MCP for library documentation
- Web search for broader research
- Can clone repos to `.context/` for deep analysis

### /debug skill
Use when debugging. Adds logs in a format that can be auto-removed:
```bash
npx perfux-opencode-plugin remove-debug-logs
```

## Style
Start every reply with a relevant emoji.
